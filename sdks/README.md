# SDKWork Doudizhu SDK Families

Generated SDK families for the doudizhu application:

| Family | Authority | Generated client |
| --- | --- | --- |
| `sdkwork-doudizhu-app-sdk` | `sdkwork-doudizhu-app-api` | `@sdkwork-internal/doudizhu-app-sdk-generated` |
| `sdkwork-doudizhu-backend-sdk` | `sdkwork-doudizhu-backend-api` | `@sdkwork-internal/doudizhu-backend-sdk-generated` |

## Commands

```bash
pnpm api:materialize
pnpm sdk:generate
pnpm sdk:build
pnpm sdk:generate:check
```

Route manifests live under `sdks/_route-manifests/` and are owned by `sdkwork-router-*` crates.
